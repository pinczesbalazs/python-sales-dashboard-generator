This project automates the processing and analysis of sales data using Python.

The application reads multiple CSV files, cleans and combines the datasets, calculates key business metrics, generates visualizations, and exports an automated management report.

## Features

- Import multiple CSV files automatically
- Merge datasets into a single data source
- Remove duplicate records
- Clean text fields and handle missing values
- Calculate business KPIs
- Identify top-performing products
- Generate monthly revenue charts
- Export automated Excel reports

## KPIs

The application calculates:

- Total Revenue
- Average Order Value
- Top Performing Product

## Technologies Used

- Python
- Pandas
- Matplotlib
- OpenPyXL
- Pathlib
